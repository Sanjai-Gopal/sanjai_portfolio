# Future-Ready Portfolio Website with Private Cloud

A modern, futuristic portfolio website showcasing professional achievements, social connections, and a private cloud storage system for certificates and personal files. Built with HTML5, CSS3, and JavaScript, featuring a responsive design and advanced animations.

## ✨ Features

### Public Portfolio Section
- **Futuristic Design**: Glitch text effects, gradient animations, and smooth transitions
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Typing animation, scroll indicators, and animated counters
- **Achievements Showcase**: Grid layout to display professional accomplishments
- **Social Integration**: Direct links to LinkedIn, WhatsApp, Instagram, GitHub, and email
- **Newsletter Subscription**: Form for visitors to stay updated

### Private Cloud Section
- **Secure Authentication**: Passcode-protected access to personal files
- **File Management**: Upload, view, download, and delete certificates and images
- **Drag & Drop Support**: Easy file uploading with visual feedback
- **File Preview**: Built-in viewers for PDFs and images
- **IndexedDB Storage**: Persistent storage with better capacity than localStorage
- **File Organization**: Sort by upload date, file type icons, and size formatting
- **Notification System**: User-friendly alerts for all actions

### Technical Highlights
- **IndexedDB Integration**: Robust client-side database for file storage
- **AOS Library**: Animate on Scroll for engaging user experience
- **Font Awesome Icons**: Professional iconography throughout
- **CSS Animations**: Custom keyframe animations for visual appeal
- **Modular JavaScript**: Organized code structure with separate files for different functionalities
